Seletores personalizados são uma funcionalidade do CSS que permite criar seletores personalizados para aplicar estilos a elementos específicos em um documento HTML. Eles são criados usando a pseudo-classe :is(), que aceita uma lista de seletores como argumento. Isso permite que você aplique estilos a vários elementos de forma mais eficiente e legível.
